\3. Fixed-point Numbers: fixed-point numbers are binary fractions, where e = 0, x = f, hence |x| = |f| < 1. We can therefore only deal with numbers that are in the interval (−1,1).

\4. An algorithm is a finite sequence of instructions for the solution of a class of problems. An algorithm’s complexity refers to the bounds on its use of memory (i.e. data) and run time (i.e. number of steps). An algorithm’s stability means the behavior of the algorithm is not sensitive with respect to the error of input parameters of the algorithm.

A mathematical problem is well-posed (well-behaved or well-conditioned) provided that its solutions enjoy the three benign properties of existence, uniqueness, and continuity with respect to the input data. A problem which is not well-posed is called ill-posed (ill-behaved or ill-conditioned).

\5. Double-precision Numbers: To increase the precision, one can use two machine registers to represent a machine number. In effect, one then embeds R(t,s) ⊂ R(2t,s), and calls x ∈ R(2t,s) a double-precision number.
